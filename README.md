# Frontend Component Library

This repository now includes a minimal React 18 + TypeScript + Tailwind CSS + Storybook setup around the `PrimaryButton` component.

## Run locally

1. Install dependencies:

   ```bash
   npm install
   ```

2. Start Storybook:

   ```bash
   npm run storybook
   ```

3. Optional local development app:

   ```bash
   npm run dev
   ```

## Useful scripts

- `npm run storybook` — start Storybook on port `6006`
- `npm run build-storybook` — generate a static Storybook build
- `npm run test` — run Vitest unit tests
- `npm run build` — type-check and build the Vite app
